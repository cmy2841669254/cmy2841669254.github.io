# 前言

阅读框架源码并不是一件容易的事情，在阅读源码前要先弄清楚自己为什么要阅读源码，以及通过读源码想达到什么目的，这个是很重要的。在阅读源码前最好对框架有一定的了解，切勿为了看源码而阅读源码，应该带着疑问去阅读，比如响应式数据变更后视图为什么也会随之变化、vue2和vue3响应式原理的区别等等。

阅读框架源码的难点主要集中在无从下手、源码内容抽象，好在vue的源码难度不算高，本文档将会以较为通俗易懂的示例来一步一步刨析Vue.js的实现。

如果你对源码感兴趣，但不知道该从何处开始看起，强烈建议阅读《Vue.js设计与实现》这本书，该书基于 Vue.js 3 深入解析内部设计细节，该书由Vue.js官方团队人员 霍春阳 撰写，如果你时常逛知乎，并且刷过vue框架相关的内容，想必你多多少少会看到以下这个人：
